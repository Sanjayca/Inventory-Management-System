## How To Use

### Back-end

#### Installation

```html
cd .\server\
```

```html
pip install -r requirements.txt
```

#### Package model changes

```html
python manage.py makemigrations
```

#### Execute the changes into the database

```html
python manage.py migrate
```

#### Run the server

```html
python manage.py runserver
```

---

## Front-end

#### Installation

```html
cd .\Client\
```

```html
npm install
```

#### Run the server

```html
npm start
```
